# cloudwatch-alarm-dashboard

This is a sample project for creating Cloudwatch alarms and Cloudwatch dashboards with various metrics

Metrics monitored:

* EC2 CPU utilization
* EC2 network in
* EC2 network out
* EC2 disk read
* EC2 disk write
* API gateway
* Backups
* Transit gateway stats
* VPN stats
* Athena stats
* ELB stats
* Lambda stats
* S3 stats

Alarms:

* High CPU (at 80%)
* Backup failure
* VPN status
